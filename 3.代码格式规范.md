# eslint：javascript 代码检测工具

# prettier：前端代码格式化工具

# stylelint：css 代码规范工具

## 安装依赖
```
  stylelint
  stylelint-config-html
  stylelint-config-prettier
  stylelint-config-standard
  stylelint-order
```

## .stylelintrc.js 配置
``` javascript
module.exports={
  "extends": "stylelint-config-standard"
}
```
